# CSGO Script (Python 3.7)

## Библиотеки
- pymem
- requests
- PySide
- beautifulsoup4

## Функционал
- bhop
- wallhack (плоховато)
- aim (нет настроек области и скорости наведения на модельку)
- автообновеление оффсетов
- GUI на Pyside
